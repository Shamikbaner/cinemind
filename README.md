# CineMind

CineMind is an AI-powered production-grade OTT streaming platform featuring personalized recommendations,scalable streaming infrastructure, and modern user experiences inspired by Netflix.
The platform focuses on personalized recommendations, scalable streaming infrastructure, modern UI/UX, and advanced machine learning systems

## Features'

- Authentication System
- Personalized Recommendations
- Smart Search
- Watch History
- Ratings & Reviews
- Watch Party
- AI-Based Recommendations
- HLS Adaptive Streaming
- Dynamic UI Personalization
- Analytics Dashboard
- Real-time Notification
- Multi-language Support
- AI Curator

## Tech Stack
